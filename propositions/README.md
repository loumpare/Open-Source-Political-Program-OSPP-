# Proposals

Proposals are organized as `{country_code}/{domain}/`. To add or modify one, see [CONTRIBUTING.md](../CONTRIBUTING.md).

## Structure

```
propositions/
├── global/          Universal proposals (any democratic system)
│   └── governance/
├── fr/              France
│   ├── economie/
│   ├── social/
│   ├── environnement/
│   ├── education/
│   └── sante/
├── us/              United States
│   ├── economy/
│   ├── social/
│   ├── environment/
│   ├── education/
│   └── health/
└── {cc}/            Any country (ISO 3166-1 alpha-2)
```
